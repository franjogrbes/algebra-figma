LESSON 7

How to Use CSS Model Effectively

When designing web layouts, understanding relationship between typography and the box model is crucial. Each text element occupies space that can be controlled padding, margin and border properties.

KEY CONCEPTS

  • Content - The actual text or content inside the element
  • Padding - Space between content and border
  • Border - The line around the padding and content
  • Margin - space outside the border, between elements

PRO TIP

Use consistent spacing values to create visual hierarchy and improve readability. Notice how this callout uses padding and left border for emphasis.
